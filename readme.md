# 🏥 Salud Friendly - Plataforma Digital de Salud

## Descripción
Plataforma web funcional para gestión de citas médicas, resultados y trámites de salud. Desarrollada como proyecto universitario.

## Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3
- **Backend:** Python con Flask
- **Base de Datos:** SQLite

## Instalación

### 1. Instalar Python
Descarga desde: https://www.python.org/downloads/
